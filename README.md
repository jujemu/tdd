## 프로젝트 소개

## 기능 요구사항

### 관리자

 - 주문 목록에 음료 추가/삭제 기능
 - 주문 목록 전체 지우기

### 키오스크
 
- 상품 리스트를 조회할 수 있다.
- 상품의 판매 상태
  - 판매중, 판매 보류, 판매 중지
- id, 상품 번호, 상품 타입, 판매 상태, 상품 이름, 가격
- 운영 시간(10~22) 외에는 주문할 수 없다.
- 주문 생성하기
- 주문 목록 총 금액 계산하기
